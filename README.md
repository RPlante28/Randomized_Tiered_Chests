## Info / Pre-reqs
This is a companion plugin I wrote to [Ayman-Isam's HungerGames Plugin](https://modrinth.com/plugin/hungergames)

This plugin relies on file structure provided by this afformentioned plugin

## What this plugin does
This plugin scans a given arena, and rolls a dice based on a table to assign different tiers to all chests within the arena
